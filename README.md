# Yuvam App

Çiftlerin Nişan, Kına, Düğün gibi organizasyonlarını kolayca yönetebileceği uygulama.

## 🚀 Özellikler
- Bütçe Takibi (Yapılan Ödemeler ve Yaklaşan Ödemelerin takibi)
- Görev takibi (Yapılması gereken rezervasyonlar , alınacak şeyler araştırılması gereken konular)

## 🛠️ Kullanılan Teknolojiler
- Flutter
- Dart
- Firebase
